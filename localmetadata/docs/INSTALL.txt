Installation

1. copy plugin bundle into the PMS plug-ins directory
located on Windows in %LOCALAPPDATA%/Plex Media Server/Plug-ins

2. restart PMS

3. configure agents priority for Personal Media Shows
Local Media Assets should be first
Local Metadata info (our plugin) should be second from the top
make sure the plugins are enabled

4. configure a new Library in PMS
in the advanced tab of the Add dialog choose Personal Media Shows
as the main agent and Plex Media Scanner as the scanner
other settings should be set to defaults
